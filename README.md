# IT-LAB-analog-keyboard

Project binnen het IT-LAB om een analoog toetsenbord te maken.

## Roadmap

### Opzoekwerk

    - Welke materialen zijn er nodig?
    - Hoe moet je werken met Arduino en programmeren in c?
    - Kennis opbouwen over elektrische circuits.

### Aankooplijst

- Microcontroller
    - [Liatris](https://splitkb.com/products/liatris?variant=43398706036995)
    - [Adafruit feather](https://www.adafruit.com/product/4884)

- 13 bit hall effect sensor
    - model: SI7213-B-00-IV
    - [datasheet](https://www.mouser.be/datasheet/3/340/1/si721x-data-sheet.pdf)

- 1ch 8bit adc i2c
    - model: ADC081C021CIMM/NOPB
    - [datasheet](https://www.ti.com/lit/ds/symlink/adc081c021.pdf)

- 8 channel i2c multiplexer
    - model: PCA9547BS,118
    - [datasheet](https://www.farnell.com/datasheets/1815500.pdf)

- Switches
    - [lekker](https://wooting.io/product/lekker-switch-l60-v2?Size=70%20pack)
